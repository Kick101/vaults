
| *Vertex*           | *Description*                                                                                              |
| ------------------ | ---------------------------------------------------------------------------------------------------------- |
| **Adversary**      | The individual, group, or entity responsible for conducting the intrusion.                                 |
| **Capability**     | The tools, malware, or techniques used in the attack (e.g., ransomware, exploits).                         |
| **Infrastructure** | The resources used by the adversary to deliver or control the attack (e.g., domains, IPs, email accounts). |
| **Victim**         | The target of the attack (individual, organization, system, or network).                                   |
