
# ✅ pfSense Learning Roadmap Checklist

## 🔰 Phase 1: Fundamentals & Installation
- [x] Understand what pfSense is and where it's used
- [x] Download and install pfSense in a VM or on bare metal
- [x] Complete the initial configuration wizard
- [x] Learn to navigate the pfSense web GUI
- [ ] Backup and restore configuration files

## 🌐 Phase 2: Networking Basics
- [ ] Review basic networking concepts (IP, subnetting, DHCP, DNS, NAT)
- [x] Configure WAN and LAN interfaces
- [x] Set up the DHCP server
- [x] Configure the DNS resolver or forwarder
- [ ] Set up static routes

## 🔐 Phase 3: Firewall & Security
- [ ] Understand the firewall rule processing order
- [x] Create LAN-to-WAN firewall rules
- [x] Create block rules (e.g., deny access to certain IPs)
- [ ] Set up aliases for IPs, networks, or ports
- [ ] Secure web interface (HTTPS, admin password, remote access)
- [x] Enable SSH with key-based login

## 🌍 Phase 4: Advanced Network Services
- [ ] Create and assign VLANs
- [x] Configure and connect to OpenVPN
- [ ] Configure WireGuard or IPSec VPN
- [ ] Set up Dynamic DNS (DDNS)
- [ ] Configure static DHCP mappings
- [ ] Set up Captive Portal for guest networks
- [ ] Explore High Availability with CARP

## 🧰 Phase 5: Add-ons & Packages
- [x] Install pfBlockerNG and configure ad/GeoIP blocking
- [ ] Install and configure Snort or Suricata (IDS/IPS)
- [ ] Set up ntopng for traffic visualization
- [ ] Install and configure Squid (web proxy)
- [ ] Configure SquidGuard for content filtering

## 🧪 Phase 6: Monitoring, Logging & Maintenance
- [ ] View and interpret system logs
- [ ] Use traffic graphs and monitoring tools
- [ ] Set up remote syslog server
- [ ] Create regular backups (manual or scheduled)
- [ ] Upgrade pfSense firmware
- [ ] Enable alerting and health monitoring

## 🧠 Phase 7: Labs & Real-World Projects
- [ ] Set up a segmented network with VLANs (e.g., IoT, Guest, LAN)
- [ ] Create a site-to-site VPN between two pfSense instances
- [ ] Build a dual-WAN failover and load-balancing setup
- [ ] Use pfBlockerNG to block categories (e.g., social media, ads)
- [ ] Simulate intrusion detection with Snort or Suricata
- [ ] Build a secure guest Wi-Fi with Captive Portal and traffic shaping
