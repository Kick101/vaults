**Secure Score** provides a **quantitative measure** of your organization’s security posture. It is part of **Microsoft Defender Exposure Management** and offers two views:

- **Microsoft Secure Score (365):** Focuses on **identity, apps, and device posture** in Microsoft 365.
- **Cloud Secure Score (Risk-Based):** Reflects your **cloud infrastructure security posture** (risk-adjusted).


>Higher scores (better) = More recommended security actions implemented.

### 🎯 **Benefits of Secure Score**

- Central dashboard for monitoring security posture
- Visualized trends and metrics
- Score comparison with industry peers
- Integration with Microsoft security tools
- Acknowledges **third-party or alternate mitigations**

### 📦 **Products Covered**

Secure Score gives recommendations for these products:

- Microsoft Defender for Office
- Exchange Online
- Microsoft Entra ID
- Microsoft Defender for Endpoint
- Microsoft Defender for Identity
- Microsoft Defender for Cloud Apps
- Microsoft Purview Information Protection
- Defender for Cloud Apps
- Microsoft Teams
- App governance
- Citrix ShareFile
- Docusign
- GitHub
- Okta
- Salesforce
- ServiceNow
- SharePoint Online
- Zoom

### ✅ **Improve Your Score**

Go to the **Recommended actions** tab to:

- View improvement suggestions
- Track status:  
    `To address`, `Planned`, `Risk accepted`, `Resolved via third party`, `Resolved via alternate mitigation`, `Completed`
- Search, filter, group by category or status
