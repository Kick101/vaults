# Device inventory list

>By default, the queue displays devices with alerts seen in the last *30 days*.

![[Pasted image 20250705202452.png]]

### 🛡️ Risk Level

- Based on active alerts + pending remediation.
- Fixing issues = lower risk.

### ⚠️ Exposure Level

- Based on missing security settings.
- Levels: Low / Medium / High.
- _No data?_ *->* Device inactive, unsupported OS, or old agent.

### ❤️ Health State

- **Active** – Reporting OK
- **Inactive** – No data >7 days
- **Misconfigured** – Sensor/connection issues

### 🧬 Antivirus Status (Win10)

- **Disabled** – Protection off
- **Not reporting** – No status
- **Not updated** – Outdated AV

# Investigate the device

**Affected devices are identified in the following areas:**

- Devices list
- Alerts queue
- Security operations dashboard
- Any individual alert
- Any individual file details view
- Any IP address or domain details view

## When you investigate a specific device, you'll see

1. **Device details:** device's domain, OS, and health state.

2. **Response actions**
	- Manage tags
	- Isolate device
	- Restrict app execution
	- Run antivirus scan
	- Collect investigation package
	- Initiate Live Response Session
	- Initiate automated investigation
	- Consult a threat expert
	- Action center

3. **Cards** (active alerts, logged on users, security assessment)

### 4. Tabs 

#### 1. Overview

##### 🚨 Active Alerts

- View total alerts from last 30 days.
    
- Grouped by:
    - **New**
        
    - **In Progress**
        
- Subcategories by **severity level**.
    
- Click alert rings to filter the list.
    

##### 👤 Logged-on Users

- Shows user logons in last 30 days.
    
- Highlights most/least active users.
    
- "See all users" → Details pane:
    
    - User type
        
    - Sign-in type
        
    - First/last seen
        

##### 🛡️ Security Assessments

- Shows:
    
    - **Exposure level**
        
    - **Security recommendations**
        
    - **Installed software**
        
    - **Detected vulnerabilities**
        
- Exposure = based on pending recommendations.

#### 2. Alerts

- **Description**
    
- **Severity**: High | Medium | Low | Informational
    
- **Status**: New | In Progress | Resolved
    
- **Classification**: Not set | False alert | True alert
    
- **Investigation state**
    
- **Category**
    
- **Assigned to**
    
- **Last activity timestamp**

#### 3. Timeline

##### 🔍 Key Features:

- **Search events** using keywords.
    
- **Filter by date** (last day, week, 30 days, or custom).
    
- **Jump to a specific moment** using timeline and arrows (e.g., auto investigations).
    
- **Export timeline data** (up to 7 days).
    
- **Event details** include:
    
    - Event type (e.g., threat detected, remediation success/failure)
        
    - Alert category (e.g., _Lateral Movement_)
        
    - Isolation context (_Application Guard_)
        
    - User actions (_SmartScreen bypass_)
        

##### 🚩 Flagging Events:

- Highlight key events
    
- Mark items for deeper investigation
    
- Build breach timelines
    

> Use the **flag icon** in the Flag column.  
> Filter with “Only Flagged Events” to review marked entries.

#### 4. Security recommendations

Generated by **Threat & Vulnerability Management** to reduce device exposure.

- View **description**
    
- Understand **associated risks** if not applied
    
- Helps prioritize remediation actions

#### 5. Software inventory

- Lists **installed software** on the device
    
- Shows **weaknesses** and **known threats**
    
- Click on software name to view details:
    
    - 🔐 Security recommendations
        
    - 🐞 Vulnerabilities
        
    - 🖥️ Devices with that software
        
    - 📊 Version distribution

#### 6. Discovered vulnerabilities

- Lists all found vulnerabilities on the device
    
- Shows:
    
    - 🔥 Severity level
        
    - 📈 Threat insights
        
- Click a vulnerability to view:
    
    - 📃 Description
        
    - 🔍 Detailed information

#### 7. Missing KBs (Knowledge Base IDs)

The Missing KBs tab lists the missing security updates for the device.
