### Threat intelligence Upload API connector

"Structured Threat Information Expression" -- *STIX*

The Threat Intelligence Upload API is *installed as part of the Microsoft Sentinel* Threat Intelligence solution, and managed like a data connector, but the upload API ingests threat intelligence into Microsoft Sentinel without the need for a data connector.

#### Prerequisites

- You must have *read and write permissions to the Sentinel workspace* to store your threat intelligence *STIX objects*.
- You must be able to *register a Microsoft Entra application*.
- Your Microsoft Entra application must be granted the Microsoft *Sentinel Contributor role* at the workspace level.

**Any of the following Microsoft Entra roles include the required permissions:**

- Application administrator
- Application developer
- Cloud application administrator

>The [default user role permissions](https://learn.microsoft.com/en-us/entra/fundamentals/users-default-permissions) allow users to create application registrations.

#### 1. Assign **Microsoft Sentinel Contributor** Role to the Application

**Purpose:** Required to use the Threat Intelligence Upload API.

**Steps:**

- Go to **Azure portal** -> **Log Analytics workspaces**
    
- Select your workspace -> **Access control (IAM)**
    
- Click **Add** -> **Add role assignment**
    
- In the **Role** tab:
    
    - Select **Microsoft Sentinel Contributor**
        
    - Click **Next**
        
- In the **Members** tab:
    
    - Set **Assign access to** -> _User, group, or service principal_
        
    - Click **Select members**
        
    - Search for your application by name (Microsoft Entra apps are not shown by default)
        
- Select your app -> **Review + assign**
    

→ Confirm that the role is assigned at the **workspace level**

![[Pasted image 20250804144541.png]]

#### 2. Configure Your TIP or Custom App

**Required values for upload API:**

- **Application (client) ID**
    
- **Microsoft Entra access token** (via OAuth 2.0)
    
- **Microsoft Sentinel workspace ID**

→ Enter these into your threat intelligence platform (TIP) or custom application configuration.

#### 3. Upload Threat Intelligence

- Use the **Upload API** to submit STIX objects  
    → Reference: Microsoft Sentinel Upload API documentation
    
- Within minutes, objects should appear in:  
    → **Microsoft Sentinel** -> **Threat intelligence** page

>Threat Intelligence Upload API uses which technology to authenticate with Microsoft Entra ID: *OAuth 2.0 authentication*
