**Automation Types**

- **Automation rules** -> Centrally manage automation of incident handling and response for multiple analytics rules.
    
- **Playbooks** -> *Predetermined sequences of actions* for advanced automation, integrating internal/external systems.

### Automation Rules

- Automatically assign incidents to personnel.
- Tag incidents for classification.
- Change status (including closure with reason and comments).
- Change severity.
- Run playbooks for complex automation.
- Apply automation to multiple analytics rules.

### Playbooks

- Collection of response/remediation actions and logic.
 - Run automatically in response to alerts/incidents.
- Triggered by analytics rules or automation rules.
- Run manually on-demand from incidents page.
- **Integration** -> Connect with internal and external systems.
