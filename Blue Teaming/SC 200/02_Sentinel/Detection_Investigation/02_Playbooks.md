### Azure Logic Apps


- Cloud-based service *for automating workflows.*
	
- Uses *Logic Apps Designer* (graphical UI) or **code view** (JSON format).
	
- **Workflow Construction**
    
    - Arrange **prebuilt components** in a visual sequence.
        
    - Option to write/edit automation as JSON logic.## **Logic App Info Sections**

- **Essentials** → Metadata like number of triggers/actions.
    
- **Summary** → Overview and quick access to Logic App Designer or trigger history.
    
- **Runs History** → Success/failure records of past executions.
### Logic Apps Connectors

- **Function**
    
    - Interface to external services (hundreds available).
        
    - Facilitate integration of events, data, and actions across various platforms, systems, and services.
        
- **Important Distinction**
    
    - **Microsoft Sentinel data connector** → Connects Sentinel to external data sources (Microsoft & non-Microsoft).
        
    - **Logic Apps connector** → Enables API-based integration with external services for automation workflows.
